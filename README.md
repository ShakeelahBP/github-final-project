A simple interest calculator script to empower micro-finance startups 
and provide opportunities to low income individuals.

## Introduction
This project hosts a Bash script that calculates simple interest based 
on user-provided principal, rate of interest, and time period.

## Simple Interest Formula
Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the script and follow the prompts:
```bash
bash simple-interest.sh
```
